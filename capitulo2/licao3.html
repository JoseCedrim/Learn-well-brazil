Sinais de reversão de faixa

Lição 3

Os sinais de reversão de faixa são usados para controlar a direção do tráfego em faixas específicas de uma estrada. Um uso comum desse controle são as faixas reversíveis que alteram o fluxo de tráfego durante diferentes horas do dia. O(s) sinal(s) em uma ou mais faixas mudam entre um X vermelho e uma seta verde.

Vermelho sólido X

Quando um X vermelho é exibido em uma faixa de direção, o motorista não entra ou permanece nessa faixa. Este sinal indica que a faixa está sendo usada pelo tráfego que se aproxima. Mova-se com segurança para uma pista com uma seta verde.

Seta verde apontando para baixo

Um motorista de frente para a seta verde apontando para baixo pode viajar nessa faixa.
